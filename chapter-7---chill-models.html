5 Chapter 7 - Chill Models

In this chapter, winter chill accumulation was analysed using the chillR package. Winter chill is an essential requirement for many temperate fruit trees, as sufficient cold exposure during dormancy is necessary for proper bud break and flowering in spring.

First, the chilling() function was applied to the Winters_hours_gaps dataset to calculate chill accumulation within a defined time period. This step demonstrates how hourly temperature data can be transformed into biologically meaningful chill units.

Next, a custom temperature-weighting model was developed using the step_model() function. By defining specific temperature ranges and assigning different weights to them, the model reflects the assumption that chill effectiveness varies across temperatures. Moderate temperatures contribute most to chill accumulation, whereas very low or high temperatures have little or no effect.

Finally, the custom model was applied and compared with an established model (GDH) using tempResponse(). This highlights how different modelling approaches can lead to different estimates of chill accumulation, depending on their underlying temperature response assumptions.

5.1 Questions

  1. Run the chilling() function on the Winters_hours_gap dataset
  2. Create your own temperature-weighting chill model using the step_model() function
  3. Run this model on the Winters_hours_gaps dataset using the tempResponse() function.

5.2 Answers

1. Running the chilling() function.

Code 
library(chillR)


out1 <- chilling(
  make_JDay(Winters_hours_gaps),
  Start_JDay = 90,
  End_JDay   = 100
)

out1

2. Little changes, but its mine!

Code 
df <- data.frame(
  lower  = c(-1000, 1, 2, 3, 4, 5, 6),
  upper  = c(    1, 2, 3, 4, 5, 6, 1000),
  weight = c(    0, 1, 2, 3, 2, 1, 0)
)

my_chill_model <- function(x) step_model(x, df = df)

3. Applying the tempResponse() function to the data set.

Code 
out3 <- tempResponse(
  make_JDay(Winters_hours_gaps),
  Start_JDay = 90,
  End_JDay   = 100,
  models = list(
    My_Chill = my_chill_model,
    GDH      = GDH
  )
)

out3

# Temperature
temp <- seq(-5, 15, by = 0.1)

# Weight
weights <- my_chill_model(temp)

# Plot
plot(
  temp, weights,
  type = "s",
  xlab = "Temperature (°C)",
  ylab = "Chill weight",
  main = "Temperature response of custom chill model"
)
