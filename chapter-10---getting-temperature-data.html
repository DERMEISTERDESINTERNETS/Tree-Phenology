10 Chapter 10 - Getting Temperature Data

10.1 Questions

  1. Choose a location of interest and find the 25 closest weather stations using the handle_gsod function
  2. Download weather data for the most promising station on the list
  3. Convert the weather data into chillR format

10.2 Answers

For the answers I computed the weather stations near the location Klagenfurt, Austria (46°37′N 14°18′E). The weather data also has a column for the coverage of the weather over the years. Therefore I selected all the years, which show a high percentage of coverage of data over the last century. Why did I work like this? At some point working through the github pages, realized, that we will use this data a lot in the upcoming chapters. Therefore I tried to make a dataframe for Klagenfurt, Austria (my location), which I can use throughout the whole semester. Therefore my code is a bit longer than in the module. It was pretty hard to get the data right. I did research on this topic and found out, that it was not common in Austria until ~1970 that the weather data is monitored and shared.

library(chillR)
library(dplyr)
library(purrr)

# Defining the variables

# Searching for stations

station_list <- handle_gsod(
  action = "list_stations",
  location = location,
  time_interval = time_interval_obs
)



# Sanity Check
head(station_list[, c("chillR_code","STATION.NAME","CTRY","Lat","Long","BEGIN","END","Distance","Overlap_years","Perc_interval_covered")], 10)

best

station_list_aux %>%
  select(chillR_code, STATION.NAME, CTRY, Lat, Long, Distance, Overlap_years, Perc_interval_covered) %>%

  head(15)

aux_codes <- station_list_aux %>%
  arrange(desc(Overlap_years), Distance) %>%
  slice(1:5) %>%
  pull(chillR_code) %>%
  as.character()

patch_weather <- handle_gsod(
  action = "download_weather",
  location = aux_codes,
  time_interval = time_interval_obs
) %>% handle_gsod()

patched_monthly <- patch_daily_temps(
  weather = klagenfurt_obs_all,
  patch_weather = patch_weather,

  max_stdev_bias = 2,

miss <- klagenfurt_filled %>%

  filter(is.na(Tmin) | is.na(Tmax)) %>%

miss

miss_blocks <- miss %>%
  arrange(Date) %>%
  mutate(gap = as.integer(Date - lag(Date)),
         new_block = ifelse(is.na(gap) | gap > 1, 1, 0),
         block_id = cumsum(new_block)) %>%
  group_by(block_id) %>%
  summarise(start = first(Date), end = last(Date), n_days = n(), .groups="drop")

miss_blocks
